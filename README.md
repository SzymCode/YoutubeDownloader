# YoutubeDownloader

Simple youtube downloader made with tkinter. You can choose file extension .mp3, .mp4 or .3gpp

## Instructions
• Download project files

• Install tkinter/pytube

```bash
pip install tkinter/pytube
```

• Run main.py
